.. _data-type-wrap:

wrap
====

.. container:: table-row

   Data type
         wrap

   Examples
         *This will cause the value to be wrapped in a p-tag coloring the
         value red:* ::

            <p style="color: red;"> | </p>

   Comment
         <...> \| </...>

         Used to wrap something. The parts on the left and right of the vertical
         line are placed on the left and right side of the value.
